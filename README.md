# Logistic-Regression-MultiDataset-Analysis
This project evaluates Logistic Regression performance across
multiple datasets including Titanic, Diabetes, and Heart Disease.

Objectives:
- Apply single ML algorithm on multiple datasets
- Compare performance differences
- Analyze dataset suitability

Algorithm Used:
- Logistic Regression

Tools:
- Python
- Pandas
- Scikit-learn
- Kaggle Notebook
